# 2D Quest Game (Unity)

## English

This project is a 2D quest game developed in Unity.

The game includes:
- Main menu and settings system
- Player movement and character controls
- Interactive actions triggered by in-game triggers
- Dialogue system
- Mini-games integrated into gameplay

The goal of the project is to create an immersive 2D adventure experience combining exploration, interaction, and puzzle-solving mechanics.

---

## Русский

Данный проект представляет собой 2D квест-игру, разработанную на Unity.

В игре реализовано:
- Главное меню и система настроек
- Передвижение и управление персонажем
- Интерактивные действия, запускаемые триггерами
- Система диалогов
- Встроенные мини-игры

Цель проекта — создание увлекательного 2D приключения, сочетающего исследование, взаимодействие с объектами и решение игровых задач.
